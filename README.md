# DeskSorting

基于JavaFX，辅助排座位的小程序  
`JDK: 1.8`

## 功能

- 导入座位表
- 导出座位表
- 对学生随机排序
- 对现有座位表进行*世龙排*
- 通过点击两名学生进行快速交换
- 自动排除导入座位表的错误

## 注意事项

1. 在程序里看到这份文档时，程序已在程序文件的目录下创建了**导入学生**的表格。 请根据指示在表格中填入名字、拼音（**全部小写，不能有空格**）、是否住校，至少2名学生。
2. **请不要导入原表格！已做更改！（删除了第8行）**
3. 导入导出界面可以用回车代表点击确定
4. 出现任何bug，请在Github上发issue（或私信）（可通过主界面左下角版权按钮直达）
